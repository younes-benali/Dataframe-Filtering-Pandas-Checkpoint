# Employee Data Analysis with Pandas

## Overview
This project demonstrates fundamental data manipulation techniques using Python's pandas library. The code creates a sample employee dataset and showcases how to extract specific data subsets using both `loc` and `iloc` methods.

The code is in :  [Dataframe-Filtering-Pandas-Checkpoint](Dataframe-Filtering.ipynb)


## Features
- Creates a structured employee dataset with 6 records
- Demonstrates data selection using `loc` (label-based selection)
- Demonstrates data selection using `iloc` (integer-based selection)
- Shows practical examples of filtering and column selection

## Dataset Structure
The dataset contains the following columns:
- **Name**: Employee name
- **Department**: Department where employee works (IT, Marketing, Sales, Finance)
- **Age**: Employee age in years
- **Gender**: Employee gender (Male/Female)
- **Salary**: Annual salary in USD
- **Experience**: Years of experience in the company
